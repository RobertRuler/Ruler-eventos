# 🍺 RULER Craft Beer - Guía Rápida

## ⚡ INICIO RÁPIDO (1 CLICK)

### Windows:
1. Doble click en `INICIAR-RULER.bat`
2. Se abre el navegador automáticamente
3. ¡Listo! 🎉

### Mac/Linux:
1. Doble click en `INICIAR-RULER.sh`
2. Se abre el navegador automáticamente
3. ¡Listo! 🎉

---

## ❌ ¿Los scripts no funcionan?

### Manual con Python:

**Windows:**
```bash
python -m http.server 8000
```

**Mac/Linux:**
```bash
python3 -m http.server 8000
```

Luego abre: `http://localhost:8000`

## 🔧 Solución de Problemas

### "Python no encontrado"
1. Descarga: https://www.python.org/downloads/
2. Durante instalación: marca "Add Python to PATH"
3. Reinicia el ordenador
4. Vuelve a intentar

### "El navegador no se abre"
- Abre manualmente: `http://localhost:8000`

### "Puerto en uso"
```bash
# Usa otro puerto (ej: 8001)
python -m http.server 8001
```
Luego: `http://localhost:8001`
