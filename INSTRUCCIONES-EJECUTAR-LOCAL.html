Ejecutar RULER en Local

⚠️ IMPORTANTE: El archivo index.html NO funciona con doble click por restricciones de seguridad del navegador. Necesitas un servidor local.

🐍 Método 1: Python (Recomendado)

Paso 1: Verifica si tienes Python instalado python --version

Si no tienes Python, descárgalo de: python.org/downloads

Paso 2: Abre CMD o Terminal en la carpeta RULER
  • Windows: Shift + Click derecho en la carpeta → "Abrir ventana de PowerShell aquí"
  • Mac/Linux: Click derecho → "Abrir terminal aquí"
Paso 3: Ejecuta este comando python -m http.server 8000

En Mac/Linux usa: python3 -m http.server 8000

Paso 4: Abre tu navegador y ve a http://localhost:8000
✅ ¡Listo! La app está funcionando en tu navegador

📦 Método 2: Node.js

Paso 1: Si tienes Node.js instalado npm install -g http-server
Paso 2: En la carpeta RULER ejecuta http-server -p 8000
Paso 3: Abre el navegador en http://localhost:8000

💻 Método 3: VS Code Live Server

Paso 1: Abre la carpeta RULER en VS Code
Paso 2: Instala la extensión "Live Server"
Paso 3: Click derecho en index.html → "Open with Live Server"
✅ Ventaja: Se recarga automáticamente al hacer cambios
❌ NO FUNCIONA:
  • Doble click en index.html
  • Arrastrar a navegador
  • Abrir directamente (file:///...)

Razón: localStorage, CORS y Supabase requieren servidor HTTP
